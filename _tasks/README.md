# Task Dispatch System

이 폴더는 **개인비서 워크스페이스**에서 이 프로젝트로 작업을 위임할 때 사용됩니다.

## 폴더 구조

```
_tasks/
├── pending/       ← 대기 중인 작업 (개인비서가 작성)
├── in-progress/   ← 진행 중인 작업 (이 프로젝트 Cursor가 이동)
└── done/          ← 완료된 작업 (보고서 포함)
```

## 워크플로우

1. `pending/`에 새 작업 지시서가 도착
2. 이 프로젝트 Cursor에게 "대기 중인 작업 확인해줘" 요청
3. 작업 시작 시 `in-progress/`로 이동
4. 작업 완료 시 `done/`으로 이동 + 보고서 섹션 작성
