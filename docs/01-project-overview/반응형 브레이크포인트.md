# 반응형 브레이크포인트

## 디바이스별 기준

### PC (디폴트)
- **너비**: > 1024px
- 모든 레이아웃의 기본 상태
- Sense Makers: 좌 75% / 우 25% 2컬럼
- 메뉴 오버레이: 화면 너비의 25%

### 태블릿
- **너비**: 768px ~ 1024px
- Sense Makers: 좌 70% / 우 30% 또는 단일 컬럼 전환
- 메뉴 오버레이: 화면 너비의 35%
- 배너 텍스트 크기 축소

### 모바일
- **너비**: < 768px
- 모든 레이아웃 단일 컬럼
- 메뉴 오버레이: 화면 너비의 80%
- 메모 사이드바: 하단 시트 또는 별도 뷰로 전환
- 터치 스와이프 우선

---

## CSS 변수 (현재 구현)

```css
/* css/variables.css 에서 관리 */
--breakpoint-tablet: 1024px;
--breakpoint-mobile: 768px;
```

## 컴포넌트별 반응형 변화

| 컴포넌트 | PC | 태블릿 | 모바일 |
|---------|-------|--------|--------|
| 헤더 높이 | 10vh | 10vh | 10vh |
| 메뉴 오버레이 너비 | 25% | 35% | 80% |
| Sense Makers 레이아웃 | 75/25 분할 | 70/30 분할 | 단일 컬럼 |
| 배너 타이틀 크기 | 48px | 36px | 28px |
| 배너 내비게이션 | 우측 하단 | 우측 하단 | 하단 중앙 |
