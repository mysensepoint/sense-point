# 프로젝트 개요

## 프로젝트명
**SensePoint** (센스포인트)

## 비전
감각의 교차점을 탐험하며 인사이트를 발견하는 웹 플랫폼

## 목적
- 디자이너가 Figma에서 디자인하기 편하도록 **레이아웃과 기능을 선행 구현**
- 오토레이아웃(Flexbox) 기반으로 피그마 호환성 확보
- PC → 태블릿 → 모바일 **반응형** 대응

---

## 기본 조건

### 오토레이아웃
- 모든 레이아웃은 **Flexbox** 기반으로 구현
- Figma의 Auto Layout과 1:1 대응되도록 설계
- 고정값보다 비율(%) 기반 레이아웃 우선

### 반응형 디바이스 기준

| 디바이스 | 너비 기준 | 비고 |
|---------|----------|------|
| **PC** (디폴트) | > 1024px | 기본 레이아웃 |
| **태블릿** | 768px ~ 1024px | 중간 레이아웃 |
| **모바일** | < 768px | 최소 레이아웃 |

> [!tip] 상세 브레이크포인트
> → [[반응형 브레이크포인트]] 참고

## 기술 스택

| 항목 | 선택 |
|------|------|
| 마크업 | HTML5 시맨틱 |
| 스타일 | CSS3 (Flexbox, Custom Properties) |
| 인터랙션 | Vanilla JavaScript |
| 폰트 | Pretendard, Playfair Display, Noto Sans KR |
| 빌드 | 없음 (정적 파일) |

## 파일 구조 (현재)

```
sense-point/
├── index.html              # 랜딩 페이지
├── sense-makers.html       # Sense Makers 페이지
├── css/
│   ├── variables.css       # 디자인 토큰
│   ├── reset.css           # CSS 리셋
│   ├── common.css          # 공통 (헤더, 메뉴)
│   ├── landing.css         # 랜딩 스타일
│   ├── sense-makers.css    # Sense Makers 스타일
│   └── responsive.css      # 반응형
├── js/
│   ├── common.js           # 메뉴 토글, 인증
│   ├── landing.js          # 배너, 물결 효과
│   └── sense-makers.js     # 메모 시스템
├── assets/                 # 이미지 에셋
└── docs/                   # ← 이 옵시디언 볼트
```
