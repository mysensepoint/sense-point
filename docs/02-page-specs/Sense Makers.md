## 철학 & 비전

### 한 문장 정의
> **사람이 사람에게 전하고자 한 감동의 기록**

### 센스 메이커스가 말하는 것
- 우리는 [**감동을 주는 순간을 담는다.**]()
- 사람이 사람에게 전하는 감동과 예술.
- 출퇴근길의 배려, 셔터를 누르는 순간, 익숙한 모든 표현과 행동들을 천천히 여유를 가지고 살펴본다.
- 알게 모르게 좋아서, 편안해서 자주 찾거나 애착이 가는 시간과 순간들 뒤에는 고유의 **메이커스들의 손길**이 지나가 있었다.
- 그 흔적들을 좀 더 또렷하게 보여질 수 있도록 **시[^1]야를 열어주고 목소리를 연결해 주는** 것이 센스 메이커스의 역할.

### 목표
이용자들이 센스 메이커스 컨텐츠를 읽으면서 **자신을 톺아보는 시간**이 자연스럽게 이어지도록 한다.

1. **호흡 조절** — 글만 나열하지 않고, 의도적인 여백과 공백으로 하나하나 곱씹을 수 있게 설계
2. **생각의 유도** — 컨텐츠를 모두 읽은 시점에 내용을 녹인 독백·질문 형태의 글로 성찰의 시간을 이끔
3. **일기로의 연결** — 하루를 생각하고, 지난 경험을 돌아보며, 내 주변이 다른 사람들의 예술로 이루어져 있음을 인지하고 경험하도록. 감동과 예술에 초점을 맞춘 일기를 이끌어냄.

### 궁극적 비전
> 서로의 영향력을 키워가는 — 우리의 예술을 담는 플랫폼

## 2-0. 인터뷰 목록 페이지 (`sense-makers-list.html`)

메뉴에서 Sense Makers 진입 시 표시되는 페이지. **인터뷰 컨텐츠 전용**.
일상 뒤에 숨겨진 메이커스들의 이야기를 카드 목록으로 보여준다.

### 레이아웃

```
┌──────────────────────────────────────────────┐
│                  헤더 (10%)                    │
├──────────────────────────────────────────────┤
│  감동의 기록                                  │
│  Sense Makers (타이틀)                        │
│  사람이 사람에게 전하고자 한 감동의 기록         │
├──────────────────────────────────────────────┤
│  [All] [Space] [Sound] [Flavor] [Craft] [Light]│
├──────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────┐ │
│  │      Featured Interview Card (가로)      │ │
│  └─────────────────────────────────────────┘ │
│  ┌──────────────────┐ ┌──────────────────┐  │
│  │  Interview Card   │ │  Interview Card   │  │
│  └──────────────────┘ └──────────────────┘  │
└──────────────────────────────────────────────┘
```

### 감각 영역 (카테고리)

| 카테고리 | 의미 | 예시 |
|----------|------|------|
| Space | 공간을 만드는 사람들 | 카페 사장, 버스 기사, 공간 큐레이터 |
| Sound | 소리를 빚는 사람들 | 재즈바 피아니스트, 사운드 디자이너 |
| Flavor | 맛과 향을 담는 사람들 | 제빵사, 꽃집 주인, 향료 연구가 |
| Craft | 손끝으로 만드는 사람들 | 가죽 장인, 목공예가, 도예가 |
| Light | 빛과 시선을 담는 사람들 | 사진작가, 조명 디자이너 |

### 구성 요소

| 요소 | 설명 |
|------|------|
| 히어로 섹션 | "감동의 기록" 태그 + "Sense Makers" 타이틀 + 철학 소개 문구 |
| 감각 영역 필터 | All / Space / Sound / Flavor / Craft / Light 탭 버튼 |
| 컨텐츠 그리드 | 2컬럼 그리드, 첫 카드는 가로 피처드 레이아웃 |
| 인터뷰 카드 | 그라데이션 비주얼 + 감각 영역 태그 + **메이커 프로필**(아바타·이름·직함) + 인터뷰 제목 + 인용구 + 메타(날짜/읽기시간) + 메모 카운트 |
| 빈 상태 | 해당 영역에 이야기가 없을 때 표시 |

## 2-1-2. 우측 영역 (25%) — 메모 사이드바

→ 상세 기능: [[메모 시스템]] 참고

### 2-1-2-1. 사이드바 헤더

| 요소 | 크기 | 설명 |
|------|------|------|
| "Memo" 텍스트 | **24pt** | 사이드바 타이틀 |
| 설명 텍스트 | **14pt** | 사용법 안내 (다음 줄) |

- 첫 메모 블럭 생성 시 → 헤더 전체 **숨김** (`display: none`)

### 2-1-2-2. 메모 블럭

#### 형태
- **포스트잇 스타일** — 따뜻한 톤 배경, 그림자, 접힌 코너
- **왼쪽 상단 접힘** (`--memo-fold-size: 14px`) — 드래그 정렬 핸들
- **입력 영역 비구분** — 경계선 없는 seamless textarea
- **빈메모/발췌 비구분** — 데이터만 구분, UI 동일
- **날짜** — 우측 하단, 10px 크기, 연.월만 (예: `26.02`)

#### 크기
| 상태 | 크기 | CSS |
|------|------|-----|
| default | 5개 블럭이 들어가는 크기 | `max-height: var(--memo-block-height)` |
| 선택(클릭) | 10줄 텍스트 가시 | `max-height: 280px` |

#### 클릭 해제 조건
1. 상단 20px 영역 클릭 (호버 시 음영 구분)
2. 다른 메모 블럭 클릭 시 자동 전환
3. 사이드바 외부 (좌측 75%) 클릭
4. ESC 키

#### 정렬 규칙
- 가장 **오래된** 블럭 → **최상단**
- 생성 시간 순으로 **하단 정렬**
- 접힌 코너 드래그로 수동 순서 변경 가능

#### 생성 조건
| 로그인 상태 | 동작 |
|------------|------|
| 로그인됨 | 메모 블럭 정상 생성 |
| 미로그인 | 메뉴 패널 열림 (로그인 유도) |

#### 생성 방법

| # | 방법 | 동작 |
|---|------|------|
| 2-1-2-2-2 | **텍스트 드래그 → 우클릭 → "Memo"** | 선택한 텍스트가 메모에 인용되어 생성, 본문 하이라이트 |
| 2-1-2-2-3 | **"+" 버튼 클릭** | 빈 메모 블럭 생성 |

#### 생성 시 동작
1. **첫 메모 생성 시** (2-1-2-2-1)
   - 헤더 전체 숨김 (타이틀 + 설명)
   - 메모 블럭이 최상단에 생성됨
2. **이후 생성 시**
   - 기존 블럭들 아래에 시간 순으로 추가

#### 텍스트 하이라이트 (2-1-2-2-2-1, 2-1-2-2-2-2)

| 상태 | 배경색 | 지속성 |
|------|--------|--------|
| 기본 (메모 생성 후) | **옅은 노란색** | 영구 적용 |
| 메모 블럭 선택 중 | **진한 노란색** | 선택 해제 시 옅은 노란색으로 복귀 |

> [!important] 하이라이트 핵심 규칙
> - 드래그하여 메모를 만든 텍스트 부분은 **영구적으로** 옅은 노란색 배경
> - 해당 메모 블럭을 **클릭/선택하는 동안만** 진한 노란색으로 변경
> - 선택 해제 시 다시 옅은 노란색으로 복귀
> - 메모 선택 시 본문의 하이라이트가 **화면 정중앙으로 자동 스크롤**

#### 수정 및 삭제
- **수정** — 클릭 시 seamless textarea에서 작성/수정
- **삭제** — 메모 블럭 **우클릭** → "삭제" 컨텍스트 메뉴 (활성/비활성 모두 가능)

#### 메모 블럭 추가 버튼 (2-1-2-2-3)

| 속성 | 값 |
|------|-----|
| 형태 | 투명색 라운드 박스 |
| 중앙 텍스트 | `+` |
| 호버 효과 | 박스 영역 음영 짙어짐 |
| 위치 (초기) | 설명 텍스트 하단 30px 간격 |
| 위치 (메모 존재 시) | 가장 마지막 메모 블럭 하단 |
