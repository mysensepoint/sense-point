## 1-2. 메인 컨텐츠 섹션 (90%)

화면 높이의 나머지 **90%** 를 차지하는 메인 영역

### 1-2-1. 배너 섹션

풀스크린 배너 캐러셀. 이미지 + 텍스트 조합.

#### 배너 슬라이드 구성 (현재)

| # | 태그 | 제목 | 설명 | CTA |
|---|------|------|------|-----|
| 1 | Discover Your Senses | 감각을 깨우는 새로운 경험 | SensePoint 소개 | Explore Now |
| 2 | Sense Makers | 생각의 결을 따라가다 | 콘텐츠 소개 | Read More |
| 3 | Square | 감각의 광장에서 만나다 | 커뮤니티 소개 | Join Square |

#### 1-2-1-1. 배너 이동 인터랙션

| # | 방식 | 설명 | 상태 |
|---|------|------|------|
| 1-2-1-1-1 | **마우스 좌우 드래그** | 좌우 드래그로 슬라이드 전환 | ✅ 구현됨 |
| 1-2-1-1-2 | **화살표 버튼** | 우측 하단에 작게 위치한 좌/우 화살표 | ✅ 구현됨 |

> [!note] 추가 고려사항
> - 터치 스와이프 (모바일/태블릿)
> - 자동 슬라이드 여부
> - 슬라이드 전환 애니메이션 종류 (현재: 수평 이동)
> - 인디케이터 도트 표시 여부

#### 배너 내비게이션 (우측 하단)
- 이전/다음 화살표 버튼
- 현재 슬라이드 / 전체 슬라이드 카운터 (예: `01 / 03`)
- **PC**: 우측 하단 고정
- **모바일**: 하단 중앙 정렬

### 1-2-2. 물결 효과 (Water Ripple)

마우스 포인터 움직임에 반응하는 **잔잔한 물결 효과**.
Canvas 기반으로 구현.

| 속성 | 값 | 비고 |
|------|-----|------|
| 구현 방식 | HTML5 Canvas | `<canvas id="rippleCanvas">` |
| 트리거 | 마우스 이동 (mousemove) | |
| 효과 | 잔잔한 물을 손으로 휘젓는 느낌 | |
| 레이어 | 배너 위에 오버레이 | pointer-events: none |
| 모바일 | 터치 이동(touchmove)으로 대체 | |

> [!note] 고도화 예정
> - 효과 강도 조절 (subtle ↔ dramatic)
> - 색상/투명도 디자이너와 협의
> - 성능 최적화 (requestAnimationFrame 주기 조절)

## 반응형 변화

### PC (> 1024px)
- 배너 텍스트: 좌측 정렬, 큰 타이틀
- 내비게이션: 우측 하단
- 물결 효과: 마우스 기반

### 태블릿 (768px ~ 1024px)
- 배너 텍스트 크기 축소
- 레이아웃 동일

### 모바일 (< 768px)
- 배너 텍스트 중앙 정렬
- 내비게이션 하단 중앙
- 터치 스와이프
- 물결 효과: 터치 기반
