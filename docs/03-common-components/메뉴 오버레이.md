# 메뉴 오버레이

> [!info] 구현 위치
> `css/common.css` — `.menu-panel`, `.menu-backdrop`
> `js/common.js` — 토글/닫기 로직

---

## 기본 동작 (1-3)

### 열기
- [[헤더]]의 **메뉴 버튼 클릭** 시 우측에서 슬라이드 인
- **모든 페이지에서 동일**하게 동작

### 닫기
- 오버레이 바깥 영역(backdrop) 클릭
- `Escape` 키 누르기
- 메뉴 버튼 다시 클릭

---

## 레이아웃 (1-3-1)

| 속성 | 값 |
|------|-----|
| 위치 | 화면 **우측**에서 슬라이드 |
| 너비 | 화면 너비의 **25%** (1-3-1-1) |
| 높이 | 전체 화면 (100vh) |
| z-index | [[헤더]]보다 **낮음** (헤더에 가려지지 않음) |
| 우측 화살표 버튼 | 메뉴가 **덮음** (1-3-1-2) |
| backdrop | 나머지 75% 영역에 반투명 검정 오버레이 |

```
┌──────────────────────┬───────────┐
│                      │           │
│    backdrop (75%)    │  메뉴(25%) │
│    (클릭 시 닫기)     │           │
│                      │           │
└──────────────────────┴───────────┘
```
