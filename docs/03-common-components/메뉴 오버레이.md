# 메뉴 오버레이

> [!info] 구현 위치
> `css/common.css` — `.menu-panel`, `.menu-backdrop`
> `js/common.js` — 토글/닫기 로직

---

## 기본 동작 (1-3)

### 열기
- [[헤더]]의 **메뉴 버튼 클릭** 시 우측에서 슬라이드 인
- **모든 페이지에서 동일**하게 동작

### 닫기
- 오버레이 바깥 영역(backdrop) 클릭
- `Escape` 키 누르기
- 메뉴 버튼 다시 클릭

---

## 레이아웃 (1-3-1) — 피그마 적용 완료

| 속성 | 값 |
|------|-----|
| 위치 | 화면 **우측**에서 슬라이드 |
| 너비 | **360px** 고정 (피그마) |
| 시작 위치 | 헤더 아래 (`top: 56px`) |
| 높이 | `calc(100% - 56px)` |
| z-index | [[헤더]]보다 **낮음** (헤더에 가려지지 않음) |
| 배경 | `#F7F4EE` (--color-menu-bg) |
| backdrop | `rgba(135, 130, 119, 0.8)` (--color-overlay-bg) |

```
┌──────────────────────┬───────────┐
│                      │           │
│    backdrop           │  메뉴     │
│    (클릭 시 닫기)      │  360px   │
│                      │           │
└──────────────────────┴───────────┘
```

## 메뉴 패널 내부 구조 (1-3-2)

```
┌─────────────────────┐
│ [로고]    [로그인]   │ ← 상단
├─────────────────────┤
│                     │
│  My Note            │
│  Sense Makers       │ ← 네비게이션
│  Square             │
│                     │
├─────────────────────┤
│ © 2026 SensePoint   │
│ contact@sensepoint  │ ← 하단 10%
│ 02-1234-5678        │
└─────────────────────┘
```

### 상단 (MenuPanel/top)
| 요소 | 스타일 | 설명 |
|------|--------|------|
| Login | text-xl, bold, --color-text-primary | 로그인/로그아웃 토글 |
| Signup | text-xl, bold, --color-text-secondary | 회원가입 링크 |

- `padding: 40px`, `gap: 24px`, flex-row
- 호버 시: accent 색상 + 하단 underline 애니메이션
- 피그마에서 메뉴 내 로고 없음 (display: none)

### 네비게이션 (MenuPanel/Nav)
| # | 메뉴명 | 링크 | 스타일 |
|---|--------|------|--------|
| 1 | **My Note** | my-note.html | text-3xl (32px), bold |
| 2 | **Sense Makers** | sense-makers-list.html | text-3xl (32px), bold |
| 3 | **Square** | square.html | text-3xl (32px), bold |

- `padding: 40px`, `gap: 24px`
- 기본: --color-text-secondary
- 호버: --color-text-primary + 하단 2px underline (--color-accent-hover `#B69BBE`)
- 현재 페이지: --color-accent

### 하단 브랜드 정보 (MenuPanel/Footer)
- `padding: 20px 40px`
- 저작권: `© 2026 SensePoint. All rights reserved.`
- 이메일: `contact@sensepoint.kr`
- 연락처: `02-1234-5678`
- 스타일: text-xs, --color-text-muted
