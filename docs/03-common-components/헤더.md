> `css/common.css` — `.header` 클래스

---

## 기본 스펙 (1-1)

| 속성 | 값 |
|------|-----|
| 높이 | 화면의 **10%** (10vh) |
| 위치 | **상시 고정** (position: fixed, top: 0) |
| 적용 범위 | **모든 페이지** |
| z-index | 최상위 — [[메뉴 오버레이]]에 가려지지 않음 |
| 배경 | 반투명 또는 블러 효과 (디자이너 협의) |

---

## 구성 요소

```
┌──────────────────────────────────────────────┐
│  [로고]                          [메뉴 버튼]  │
└──────────────────────────────────────────────┘
```

### 좌측 — 로고 (1-1-1 좌측)
- `SP` 아이콘 + `SensePoint` 텍스트
- 클릭 시 `index.html` (랜딩 페이지)로 이동
- HTML: `<a href="index.html" class="header__logo">`

### 우측 — 메뉴 버튼 (1-1-1 우측)
- 햄버거 아이콘 (3줄 → X 전환 애니메이션)
- 클릭 시 [[메뉴 오버레이]] 토글
- HTML: `<button class="header__menu-btn" id="menuToggle">`
