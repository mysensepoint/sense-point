# 간격과 레이아웃

> [!info] 구현 위치
> `css/variables.css` — Spacing, Layout, Radius 변수

---

## 간격 스케일 (Spacing)

| 토큰 | 값 | 용도 예시 |
|------|-----|---------|
| `--space-xs` | 4px (0.25rem) | 아이콘 간격 |
| `--space-sm` | 8px (0.5rem) | 인라인 요소 간격 |
| `--space-md` | 16px (1rem) | 기본 간격 |
| `--space-lg` | 24px (1.5rem) | 섹션 내부 패딩 |
| `--space-xl` | 32px (2rem) | 섹션 간 간격 |
| `--space-2xl` | 48px (3rem) | 큰 섹션 간격 |
| `--space-3xl` | 64px (4rem) | 페이지 레벨 간격 |

---

## 레이아웃 비율

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--header-height` | 10vh | [[헤더]] 높이 |
| `--content-height` | 90vh | 메인 컨텐츠 높이 |
| `--menu-width` | 25% | [[메뉴 오버레이]] 너비 |
| `--sidebar-width` | 25% | [[Sense Makers]] 메모 사이드바 |
| `--main-width` | 75% | [[Sense Makers]] 본문 영역 |

---

## 레이아웃 패턴

### 전체 페이지 구조
```
┌──────────────── 100vw ────────────────┐
│            헤더 (10vh)                 │
├───────────────────────────────────────┤
│         메인 컨텐츠 (90vh)             │
└───────────────────────────────────────┘
```

### Sense Makers 2컬럼
```
┌──── 75% (--main-width) ────┬── 25% (--sidebar-width) ──┐
│                            │                           │
│       아티클 본문           │       메모 사이드바         │
│                            │                           │
└────────────────────────────┴───────────────────────────┘
```

---

## 라운드(Radius)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--radius-sm` | 4px | 작은 버튼, 태그 |
| `--radius-md` | 8px | 카드, 메모 블럭 |
| `--radius-lg` | 12px | 패널 |
| `--radius-xl` | 16px | 큰 카드 |
| `--radius-full` | 9999px | 원형 버튼 |

---

## Z-Index 레이어

| 토큰 | 값 | 요소 |
|------|-----|------|
| `--z-ripple` | 0 | 물결 효과 캔버스 |
| `--z-base` | 1 | 일반 컨텐츠 |
| `--z-menu-overlay` | 90 | 메뉴 backdrop |
| `--z-menu` | 95 | 메뉴 패널 |
| `--z-header` | 100 | 헤더 |
| `--z-modal` | 200 | 모달 (추후) |

---

## 트랜지션 (Transition)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--transition-fast` | 150ms ease | 호버, 포커스 |
| `--transition-base` | 300ms ease | 기본 전환 |
| `--transition-slow` | 500ms ease | 페이드 인/아웃 |
| `--transition-menu` | 400ms cubic-bezier | 메뉴 슬라이드 |
