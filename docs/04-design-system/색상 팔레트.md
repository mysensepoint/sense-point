# 색상 팔레트

> [!info] 구현 위치
> `css/variables.css` — `:root` 내 컬러 변수

---

## 배경 (Background)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--color-bg` | `#0a0a0a` | 메인 배경 |
| `--color-bg-secondary` | `#111111` | 보조 배경 |
| `--color-surface` | `#1a1a1a` | 카드/패널 배경 |
| `--color-surface-hover` | `#252525` | 호버 시 표면 |

## 텍스트 (Text)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--color-text-primary` | `#f5f5f5` | 메인 텍스트 |
| `--color-text-secondary` | `#999999` | 보조 텍스트 |
| `--color-text-muted` | `#666666` | 비활성 텍스트 |
| `--color-white` | `#ffffff` | 순백 |

## 액센트 (Accent)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--color-accent` | `#c8a96e` | 브랜드 포인트 (골드) |
| `--color-accent-hover` | `#d4b87a` | 액센트 호버 |

## 기능 컬러 (Functional)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--color-highlight-light` | `rgba(255, 235, 153, 0.25)` | 메모 하이라이트 기본 (옅은 노란색) |
| `--color-highlight-active` | `rgba(255, 210, 60, 0.5)` | 메모 하이라이트 선택 중 (진한 노란색) |
| `--color-overlay-bg` | `rgba(0, 0, 0, 0.4)` | 메뉴 backdrop |
| `--color-border` | `#2a2a2a` | 구분선 |

## UI 컬러

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--color-menu-bg` | `#111111` | 메뉴 패널 배경 |

---

## 그림자 (Shadow)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.3)` | 작은 요소 |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.4)` | 카드, 메모 블럭 |
| `--shadow-lg` | `0 8px 24px rgba(0,0,0,0.5)` | 오버레이, 패널 |
