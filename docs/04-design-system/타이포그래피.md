# 타이포그래피

> [!info] 구현 위치
> `css/variables.css` — 폰트 및 크기 변수 정의

---

## 폰트 패밀리

| 용도 | 폰트 | CSS 변수 |
|------|------|---------|
| 본문 (Primary) | Pretendard | `--font-primary` |
| 디스플레이 | Playfair Display | `--font-display` |
| 한글 보조 | Noto Sans KR | (폴백) |

### 폴백 스택
```css
--font-primary: 'Pretendard', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
--font-display: 'Playfair Display', Georgia, serif;
```

---

## 텍스트 크기 스케일

| 토큰 | 값 | rem | 용도 예시 |
|------|-----|-----|---------|
| `--text-xs` | 12px | 0.75rem | 캡션, 메타 정보 |
| `--text-sm` | 14px | 0.875rem | 설명 텍스트, 사이드바 |
| `--text-base` | 16px | 1rem | 본문 기본 |
| `--text-lg` | 18px | 1.125rem | 본문 강조 |
| `--text-xl` | 20px | 1.25rem | 소제목 |
| `--text-2xl` | 24px | 1.5rem | 섹션 제목, 메모 타이틀 |
| `--text-3xl` | 32px | 2rem | 페이지 제목 |
| `--text-4xl` | 40px | 2.5rem | 배너 타이틀 |
| `--text-5xl` | 56px | 3.5rem | 히어로 타이틀 |

---

## 주요 사용처

| 요소 | 폰트 | 크기 | 굵기 |
|------|------|------|------|
| 배너 타이틀 | Display | --text-5xl | 700 |
| 배너 설명 | Primary | --text-lg | 300 |
| 아티클 제목 | Display | --text-3xl | 600 |
| 아티클 본문 | Primary | --text-base | 400 |
| 메모 사이드바 타이틀 "Memo" | Primary | **24pt** | 600 |
| 메모 사이드바 설명 | Primary | **14pt** | 400 |
| 메뉴 항목 | Primary | --text-xl | 400 |
| 브랜드 정보 (메뉴 하단) | Primary | --text-xs | 400 |
