# 메모 블럭 기능정의

## 메모 블럭 정의

### 형태
1. 크기
	1. dfault : 최대 5개의 블럭을 정렬 할 수 있는 크기 (사용자 임의 변경 불가)
	2. 클릭 시 : 10줄의 텍스트를 볼 수 있는 크기로 늘어남 (사용자 임의 변경 불가)
		1. 클릭 해제
			1. 상단으로 부터 아래로 20px 정도의 범위를 클릭 시 클릭 해제
				1. 해당 부분 마우스 호버 시 옅은 음영으로 영역 구분 
			2. 다른 메모 블럭 클릭 시 클릭 해제 및 클릭 메모 블럭 클릭 상태 활성화
			3. 우측 25%의 메모 영역 외 영역 클릭 시 클릭 해제
			4. 클릭 해제 시 dfault 값으로 복귀
2. 모양
	1. 포스트잇이 붙어져 있는 것이 직관적으로 연상되는 모양
	2. 왼쪽 상단 부가 아주 조금 접혀있음
		1. 접혀있는 부분을 클릭 한 상태에서 원하는 위치로 드래그하여 블럭 순서 변경이 가능
	3. 입력 영역이 구분되어 보이지 않음
	4. 빈메모, 발췌는 데이터로는 구분하여 처리하지만, 사용자에게는 구분하여 보이지 않음.
	5. 날짜/시간 데이터가 사용 메모 블럭에 나타날 때에는 우측 최하단에 매우 작게 연,월만 간략하게 표기됨

### 생성 방법
1. 생성 조건
	1. 로그인이 되어 있을 시 메모 블럭이 생성 될 경우
		1. 메모 블럭 생성
	2. 로그인이 되어 있지 않을 시 메모 블럭이 생성 될 경우
		1. 로그인 사이트로 이동
2. 방법 1. 텍스트 드래그 → 우클릭 → "Memo"
		1. 본문(좌측 75%)에서 텍스트 **드래그** 선택
		2. **우클릭** → 커스텀 컨텍스트 메뉴 표시
		3. **"Memo"** 항목 클릭
		4. 우측 사이드바에 **메모 블럭 생성** (선택한 텍스트 인용 포함)
		5. 본문의 드래그한 텍스트 부분에 **옅은 노란색 배경** 적용 (영구)
		6. 클릭 시 컨텐츠 드래그 내용이 화면 정중앙으로 위치하도록 컨텐츠 스크롤 자동 이동
3. 방법 2. "+" 버튼 클릭
	1. 사이드바 하단의 **"+" 버튼** 클릭
	2. 첫번째 메모가 생성되기 전까지 생성 블럭 버튼은 '텍스트를 드래그한 후 우클릭하여 메모를 추가하거나, 아래 + 버튼을 눌러 빈 메모를 생성하세요.' 의 하단 30px 공백을 두고 위치한다.
	3. **빈 메모 블럭** 생성
	4. 본문 하이라이트 없음

### 수정 및 삭제 방법
1. 수정
	1. 클릭 시 수정 및 작성 가능
2. 삭제
	1. 해당 메모 블럭 영역 우클릭 시 삭제 가능(활성/비활성 모든 상태에서 가능)
