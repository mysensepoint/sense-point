# 봄이 오려나

출근길에 바람이 달라진 걸 느꼈다. 아직 춥긴 한데 어딘가 부드러워진 바람. 해가 뜨는 시간도 조금씩 빨라지고 있다. 겨울이 끝나가고 있구나.

계절이 바뀌면 나도 조금 바뀌고 싶다는 생각이 든다. 겨울 동안 웅크리고 있던 것들을 좀 펴고 싶다. 운동을 다시 시작한다든가, 새로운 걸 배워본다든가. 계절 탓을 하는 건지 진심인 건지 모르겠지만 어쨌든 뭔가 바뀔 것 같은 느낌이 나쁘지 않다.

퇴근하고 집 앞 공원을 지나는데 나무에 뭔가 조금 돋아나려는 흔적이 보였다. 아직 꽃은 아니고 그냥 작은 혹 같은 것. 그걸 보니까 나도 그런 상태인 것 같아서 웃었다. 아직은 아닌데, 곧.
