# 새로운 시작

오늘 새 프로젝트의 킥오프 미팅이 있었다. 기대 반 두려움 반. 새로운 걸 시작할 때마다 느끼는 이 묘한 긴장감이 싫지 않다. 예전에는 그냥 떨리기만 했는데, 요즘은 이 떨림이 살아 있다는 증거라고 느낀다.

팀원들과 자기소개를 했다. 다들 다른 배경, 다른 생각. 아직은 서로를 모르지만 몇 달 뒤에는 어떤 관계가 되어 있을까. 사람과 사람이 만나서 뭔가를 같이 만들어간다는 것 자체가 흥미롭다.

집에 와서 노트에 이번 프로젝트에서 내가 얻고 싶은 걸 적어봤다. 실력도 중요하지만, 이번에는 "과정에서 즐거움을 느끼는 것"을 목표로 하고 싶다. 결과에만 집착했던 예전과는 다르게 해보자.
