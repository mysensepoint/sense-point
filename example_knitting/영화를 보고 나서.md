# 영화를 보고 나서

오늘 혼자 영화관에 갔다. 제목은 잊어버렸는데 어떤 여자가 회사를 그만두고 혼자 여행을 떠나는 이야기였다. 영화 자체는 보통이었는데 끝나고 나오면서 이상하게 울컥했다.

아마 내가 못 하는 걸 하는 사람을 봐서 그런 것 같다. 다 관두고 떠나고 싶다는 생각, 한 번도 안 해본 건 아니다. 근데 현실에서는 월세, 대출, 부모님 생각에 한 발짝도 못 움직인다. 용기가 없는 건지 책임감이 있는 건지 모르겠다.

돌아오는 길에 생각했다. 모든 걸 놓고 떠나지 않더라도 일상 안에서 작은 탈출구를 만들 수 있지 않을까. 한 달에 하루는 완전히 다른 일을 해본다든가. 여행이 아니어도 된다. 루틴을 깨는 것 자체가 여행이 될 수 있을 것 같다.
