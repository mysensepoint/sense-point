# 전시를 다녀와서

주말에 미술관에 다녀왔다. 사진작가의 전시였는데, 전부 "일상"을 찍은 사진이었다. 식탁 위의 그릇, 빨래 널린 베란다, 새벽의 편의점. 특별한 피사체가 하나도 없는데 왜 이렇게 아름답게 느껴질까.

작가 노트에 이렇게 적혀 있었다. "매일 보는 것을 처음 보는 것처럼 바라보는 연습을 합니다." 이 한 문장이 전시 전체를 설명했다.

나도 매일 같은 것을 보면서 산다. 같은 방, 같은 출근길, 같은 사무실. 이걸 "처음 보는 것처럼" 볼 수 있다면 일상이 달라질 것 같다. 익숙함에 가려진 것들을 다시 발견하는 눈. 이게 감각을 높인다는 거구나.
