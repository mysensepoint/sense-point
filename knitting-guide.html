뜨개질

흩어진 감각을 하나의 실로 꿰다

일상의 조각들—아침의 감각, 스치는 고민, 작은 기쁨—을
서사적으로 연결해 '나'를 더 깊이 이해하는 기능입니다.

시작하기

뜨개질이란?

노트를 단순히 모으는 것이 아니라, 하나의 이야기로 엮는 것입니다.

실 (Thread)

여러 노트를 관통하는 하나의 서사적 흐름입니다. 예를 들어 "아침의 감각 → 커피의 온기 → 바람의 촉감"을 감각의 흐름이라는 실로 엮을 수 있습니다.

코 (Stitch)

실 안에서 각 노트가 차지하는 자리입니다. 같은 노트라도 실마다 다른 관점에서 바라볼 수 있습니다. 코마다 "이 노트를 이 맥락에서 어떻게 보는가"를 적을 수 있습니다.

실마리 (Clue)

노트 속 특정 문장이나 키워드에 심는 표지입니다. 실마리를 심어두면, 시스템이 같은 실마리를 가진 노트들을 자동으로 연결해줍니다.

세 가지 뜨개질 방법

직접 엮거나, 시스템이 제안하거나, 둘을 합치거나.

01

직접 뜨개질

내 손으로 감각을 엮다

새로운 실을 만들고, 원하는 노트를 하나씩 골라 순서대로 꿰어 넣습니다. 각 코(노트)에 "이 맥락에서 이 노트가 의미하는 것"을 적어 관점을 기록할 수 있습니다.

1 뜨개질 탭에서 "+ 새 실" 클릭
2 실 이름과 설명 입력
3 "+ 노트 추가"로 노트를 골라 꿰기
4 각 코에 관점(Perspective) 메모 남기기
02

자동 뜨개질

시스템이 감각의 연결을 발견하다

노트에 실마리를 심어두면, 같은 실마리를 공유하는 노트들이 자동으로 묶여 실이 제안됩니다. 또한 키워드 분석을 통해, 비슷한 감각과 고민을 담은 노트들의 연결도 자동으로 발견합니다.

1 노트 작성 중 텍스트를 선택하고 "실마리로 지정"
2 실마리에 이름 부여 (예: "감각의 흐름")
3 뜨개질 탭의 "자동 제안" 영역에서 추천 실 확인
4 "이 실 만들기"로 자동 생성
03

이어 뜨개질

발견된 연결을 내 것으로 만들다

자동으로 생성된 실을 그대로 두지 않고, 직접 수정할 수 있습니다. 노트를 추가하거나 빼고, 순서를 바꾸고, 관점을 덧붙이면 자동 실이 "편집됨"으로 바뀝니다. 시스템이 발견한 연결에 나만의 해석을 더하는 과정입니다.

1 자동 생성된 실을 열기
2 코(노트)의 순서를 ▲▼ 버튼으로 재배치
3 새 노트를 추가하거나 불필요한 코 제거
4 각 코에 나만의 관점 작성

왜 뜨개질인가?

일상의 감각을 기록하는 것만으로는 부족합니다.
감각과 감각 사이를 연결할 때, 비로소 '나'가 보입니다.

단순 기록을 넘어서

하루의 감각, 생각, 고민을 개별 노트로 기록하는 것은 첫 걸음입니다. 뜨개질은 이 조각들을 하나의 서사로 엮어, 흩어진 감각 속에서 나만의 패턴을 발견하게 해줍니다.

같은 감각, 다른 시선

하나의 노트가 여러 실에 동시에 꿰어질 수 있습니다. "커피의 온기"라는 노트가 감각의 흐름에서는 촉각의 관점에서, 일상의 기쁨에서는 작은 행복의 관점에서 다르게 읽힙니다.

나에 대한 감각이 깊어지다

감각을 엮다 보면, 내가 어떤 것에 반응하고 어떤 고민을 반복하며 어디에서 기쁨을 느끼는지 보이기 시작합니다. 뜨개질은 '나'라는 주체에 대한 감각을 풍부하게 만드는 도구입니다.

실 하나의 예시

"감각의 흐름"이라는 실이 어떻게 노트를 엮는지 살펴보세요.

감각의 흐름

아침의 감각

"아침마다 이 순간을 기록하다 보면, 내가 어떤 사람인지 조금씩 더 선명해지는 것 같다"

관점: 하루의 시작에서 만나는 가장 원초적인 감각들

커피의 온기

"이 감각들의 교차점에 서 있을 때, 나는 단순히 커피를 마시는 것이 아니라 '지금 이 순간을 느끼는 나'를 인식한다"

관점: 여러 감각이 동시에 교차하는 순간

바람의 촉감

"이 미묘한 차이들을 알아채는 것이, 일상의 감각을 높이는 일이다"

관점: 같은 자극에서 미세한 차이를 알아채는 감수성

습관의 힘

처음엔 귀찮았는데 이제 안 하면 찝찝하다.

나를 바꾸는 건 거창한 결심이 아니라 작은 습관이라는 걸 실감한다. 2주 전의 나와 오늘의 나는 분명히 다르다.

이번 달을 돌아보며

체감은 "아무것도 안 했다"인데 실제로는 나름대로 움직이고 있었네. 안 그러면 시간이 그냥 흘러가버린다.

내 시간이 어디에 쓰였는지, 내 마음이 어디를 향했는지.

어제 쓴 글을 다시 읽으며

하루 지나니까 느낌이 다르다. 어제는 세상이 다 잘못된 것 같았는데 오늘 읽으니 내가 좀 예민했구나.

감정이 그만큼 순간적이라는 뜻이기도 하고, 하루만 지나도 관점이 바뀔 수 있다는 뜻이기도 하다.
